# pipeantenna

## Usage

```
pipeantenna ~/go/src/github.com/ashmeet28/pipeantenna/web_pages/index.html /mnt/t/pipeantenna_https_crt /mnt/t/pipeantenna_https_key /mnt/t/pipeantenna_auth_key_file
```

```
openssl req -x509 -newkey rsa:4096 -noenc -keyout /mnt/t/pipeantenna_https_key -out /mnt/t/pipeantenna_https_crt
```
